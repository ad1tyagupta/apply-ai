# Raw Uploads

Place original user uploads here.

Codex should give the user the full absolute path to this folder before upload. After files are added, run:

```bash
npm run uploads:check
```

Then confirm the detected file list with the user before ingesting.

Examples:

- old CVs
- cover letters
- background notes
- about-me documents
- project-history documents
- screenshots or scans

Do not upload transcripts, certificates, or mark sheets here for normal use. Summarize those in text instead if they matter for the search.
