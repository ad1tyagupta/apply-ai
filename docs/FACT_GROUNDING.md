# Fact Grounding

Every generated artifact in this repository should be auditable.

## Rules

1. Raw uploads are the source of truth.
2. Normalized Markdown files exist only to make those uploads easier to use.
3. Wiki pages must cite or clearly trace back to normalized documents.
4. CVs and cover letters may only use facts that exist in source-backed material.
5. When in doubt, leave the claim out.

## Unsafe Behaviors

- inferring a metric from vague wording
- upgrading working knowledge into fluency
- converting interests into professional experience
- inventing dates or employers

## Safe Behaviors

- quoting a date range that appears in a CV
- rephrasing a documented project into clearer resume language
- choosing the strongest factual example from uploaded materials
