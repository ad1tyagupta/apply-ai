# Profile Layer

Use the profile layer to keep evaluation grounded in user evidence.

## Files To Reuse

- `config/user-preferences.yml`
- `config/profile.yml`
- `profile/wiki/*`
- `profile/target-recommendations.md`

## Rules

- trust uploaded source documents first
- use the wiki for reusable summaries only when it stays source-backed
- treat accepted companies and role families as editable preferences
