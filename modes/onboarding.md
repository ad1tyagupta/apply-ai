# Onboarding Mode

When the workspace is still generic:

1. ask for geography
2. explain scoring modes and let the user choose one
3. ask the user to upload documents into `intake/raw/`
4. convert uploads into Markdown
5. build the profile wiki
6. generate company and role recommendations
7. let the user refine or add companies
8. build `portals.yml`
9. move into scanning and evaluation
